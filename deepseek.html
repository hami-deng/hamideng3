2. 数据爬取

为了分析 DeepSeek 爆火后的新闻报道数据，我利用 Python 的 Selenium 库编写了一个自动化程序，在火狐浏览器上通过百度搜索爬取相关新闻标题。以下是爬取过程的主要步骤：

  1. 环境搭建：安装 Python 的 Selenium 库和 Firefox 浏览器驱动（GeckoDriver）。
  2. 搜索关键词：在百度搜索框中输入关键词“DeepSeek 最新消息”，并点击搜索按钮。
  3. 爬取新闻标题：使用 Selenium 定位搜索结果页面的新闻标题元素，提取每条新闻的标题。
  4. 保存数据：将爬取的新闻标题保存到本地 CSV 文件中（deepseek_news.csv）。
from selenium import webdriver
from selenium.webdriver import FirefoxOptions
from selenium.webdriver.common.by import By
import pandas as pd

# 配置浏览器
opts = FirefoxOptions()
driver = webdriver.Firefox(options=opts)

# 访问百度搜索
driver.get("https://www.baidu.com")
search_box = driver.find_element(By.NAME, "wd")
search_box.send_keys("DeepSeek 最新消息")
search_box.submit()

# 爬取新闻标题
titles = []
for _ in range(5):  # 爬取 5 页
    results = driver.find_elements(By.CSS_SELECTOR, "h3.t")
    for result in results:
        titles.append(result.text)
    next_button = driver.find_element(By.XPATH, "//a[@class='n']")
    next_button.click()

# 保存数据
df = pd.DataFrame(titles, columns=["标题"])
df.to_csv("deepseek_news.csv", index=False)

3. 词云图制作

为了直观展示新闻标题中的关键词，我使用 Python 的 jieba 和 wordcloud 库制作了词云图。以下是制作过程的主要步骤：

  1. 数据清洗：对新闻标题进行分词，去掉标点符号和停用词。
  2. 生成词云：将分词后的结果拼接成一个字符串，作为词云图的输入。
  3. 保存和展示：将生成的词云图保存为 PNG 文件（wordcloud.png）。
from wordcloud import WordCloud
import jieba
import matplotlib.pyplot as plt

# 读取数据
titles = pd.read_csv("deepseek_news.csv")["标题"].tolist()

# 分词
text = " ".join(jieba.lcut(" ".join(titles)))

# 生成词云
wordcloud = WordCloud(font_path="simhei.ttf", width=800, height=400, background_color="white").generate(text)

# 保存词云图
wordcloud.to_file("wordcloud.png")

4. 情感分析

为了分析新闻标题的情感倾向，我使用 Python 的 SnowNLP 库对每条新闻标题进行情感分析。以下是分析过程的主要步骤：

  1. 情感得分计算：使用 SnowNLP 计算每条新闻标题的情感得分。
  2. 情感分类：根据情感得分将新闻标题分类为“积极”、“中性”或“消极”。
  3. 统计结果：统计每种情感分类的数量，并生成饼图展示情感分布。
from snownlp import SnowNLP

# 情感分析
sentiment_scores = [SnowNLP(title).sentiments for title in titles]

# 分类
sentiment_labels = ["积极" if score >= 0.6 else "消极" if score <= 0.4 else "中性" for score in sentiment_scores]

# 统计结果
sentiment_counts = pd.Series(sentiment_labels).value_counts()

# 生成饼图
plt.pie(sentiment_counts, labels=sentiment_counts.index, autopct='%1.1f%%')
plt.title("情感分析结果")
plt.savefig("sentiment_pie_chart.png")
